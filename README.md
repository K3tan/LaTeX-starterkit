> Sylvain METAYER

## Prerequisites

- `texlive-full`
- `python-pygments`
- `make`

## Build

- `make build`

The PDF will be generated in `main.pdf`

## Tags

- `make tag`

## Travis CI

A basic travis CI build is provided.

This will generate a PDF from sources on every tag on master.

PDF will be published in Github Release on every tags.
